6 RESERVED JS KEYWORDS AND WHAT THEY DO


`let`:
`let` is used to declare block-scoped variables. Variables declared with let are scoped to the block,
statement, or expression on which it is used.trations.

`const`:
`const` is used to declare variables that cannot be reassigned. The value of a constant variable remains the same throughout its scope.


`function`:
`function` is used to declare a function, which is a block of reusable code that performs a specific task.


`else`, `if`:
`if` and `else` are used for conditional execution of code. They allow you to execute different blocks of code
based on whether a specified condition evaluates to true or false.


`for`, `while`:
`for` and `while` are used to create loops, which repeatedly execute a block of code as long as a specified condition is true.


`return`:
`return` is used inside a function to return a value to the caller. It terminates the function's execution and sends the
specified value back to where the function was called.
