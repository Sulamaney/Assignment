THERE ARE SEVERAL WAYS TO DISPLAY JAVASCRIPT CODE, DEPENDING ON THE CONTEXT AND PURPOSE. HERE ARE SIX COMMON METHODS:


Inline Script Tags:
You can embed JavaScript directly within HTML using script tags. This method is commonly used for small scripts or quick demonstrations.

External JavaScript Files:
For larger scripts or scripts that are reused across multiple pages, it's better to place JavaScript code in external files with a .js extension and
include them in HTML using the script tag.


Using Browser Developer Tools:
Most modern web browsers come with built-in developer tools that allow you to inspect, debug, and modify JavaScript code directly
within the browser. You can access these tools by right- clicking on a webpage and selecting "Inspect" or by pressing F12.\


Console Output:
You can use console.log() to display JavaScript output directly in the browser console. This method is useful for debugging and
logging messages or variables.


Using alert() Function:
The alert() function displays a dialog box with a specified message and an OK button. It's primarily used for debugging or displaying
important information to the user.


Outputting to HTML Elements:
You can dynamically update HTML content using JavaScript to display messages or results within specific elements on a webpage.
