Ternary Operator:

Syntax: condition ? expression1 : expression2

The ternary operator is a concise way to write simple conditional statements.
It evaluates a condition and returns one of two expressions depending on whether the condition is true or false.
It is primarily used for assigning values based on a condition within an expression.
Suitable for simple conditional assignments but can become hard to read for complex conditions.



If-Else Statements:

Syntax: if (condition) { // code block } else { // code block }

If-else statements are fundamental to programming languages.
They allow for the execution of different blocks of code based on whether a condition evaluates to true or false.
Suitable for handling complex conditions and multiple branches of execution.
Offers greater flexibility compared to the ternary operator for handling complex scenarios.



Switch Statements:

Syntax:
switch (expression) { case value1: // code block break; case value2: // code block break; default: // code block }

Switch statements allow the selection of one of many code blocks to be executed.
They compare the expression value to multiple cases and execute the code block associated with the first matching case.
Suitable for scenarios with a single expression to be evaluated against multiple possible values.
Provides a clearer structure compared to nested if-else statements when handling multiple conditions against a single value.
